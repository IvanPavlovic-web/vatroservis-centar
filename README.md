## About the Website
Official website for Vatroservis Centar, a company providing professional services in fire safety, occupational safety, risk assessments, and office supplies.

## Demo

You can view the live demo here: [Vatroservis Centar Demo](https://ivanpavlovic-web.github.io/sweetForma/)
Note: The demo runs as a static website and requires no backend server.

## Features

- **Responsive Design** - Optimized for all devices and screen sizes  
- **Service Showcase** - Detailed presentation of fire safety and occupational safety services
- **Visual Gallery & Licenses** - Highlight key products, certifications, and authorizations
- **FAQ Section** - Quick access to important client information
- **Direct Contact** - Easy phone call or email options, no forms required
- **Lightweight & Fast Loading** - Optimized for performance on all devices

## Installation

### Method 1: Clone with Git
```bash
git clone https://github.com/IvanPavlovic-web/vatroservis-centar.git
cd vatroservis-centar
```
---
### Method 2: Download ZIP
1. Click the Code button and select Download ZIP
2. Extract the ZIP file to your preferred location
3. Navigate to the project folder
4. Open index.html, no additional installation required as it runs directly in the browser
